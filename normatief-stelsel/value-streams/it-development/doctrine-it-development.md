# Doctrine IT Development — verwijzing

De volledige doctrine voor de IT-development waardestroom staat in:

- `artefacten/0-governance/doctrine-it-development.md`

Daar is beschreven:

- wat de IT-development value stream is;
- welke fases (Trigger t/m Deployment) worden onderscheiden;
- welke doelen, activiteiten en artefacten per fase horen;
- hoe agents zich in deze waardestroom gedragen.

Dit bestand is een aanvullende verwijzing binnen de `governance/doctrine/`
folder, zodat alle doctrines en waardestromen op één plek terug te vinden zijn.
